# Scenario 5: AI 톤앤매너 텍스트 마스터 (AI Writing Style Transformer & Summarizer)

## 💡 개요
사용자가 자유롭게 작성한 날 것의 생각이나 러프한 메모를 텍스트 스타일 변경 엔진(Gemma 4)을 통해 비즈니스 보고서 톤, 인스타그램 감성 피드 톤, 엄격한 법률적 톤, 따뜻한 안부 메일 등 5가지 이상의 서로 다른 개성적인 스타일로 즉시 전환하고, 긴 복합 문서를 한눈에 들어오는 불릿 포인트 요약본으로 다듬어 주는 생산성 보조 애플리케이션입니다.

---

## 🛠️ 주요 기술 스펙
- **Few-Shot Prompting**: 각 스타일별(예: 격식체, SNS 감성체, 유머러스체 등) 대표적인 출력 예시 세트(Few-shot)를 동적으로 프롬프트에 조립하여 Gemma 4의 톤앤매너 재현 성능을 극대화합니다.
- **다국어 요약 알고리즘**: 한국어 및 영어 등 다양한 다국어 텍스트의 핵심 주제와 뉘앙스를 왜곡 없이 보존하며 요약 길이를 슬라이더(UI Slider)로 실시간 조절(짧게 / 보통 / 자세히) 가능한 지능형 요약 파이프라인.

---

## 🚀 아키텍처 및 데이터 흐름
```text
사용자 텍스트 입력 ("오늘 회의 끝남 대박 피곤함 보고서 써야되는데 귀찮")
  ➡️ 스타일 선택 슬라이더 / 칩 탭 (예: "비즈니스 격식")
  ➡️ Gemma 4 추론 (선택 스타일 템플릿 프롬프트 적용)
  ➡️ 변환된 정제 텍스트 출력:
     "금일 진행된 회의가 모두 종료되었습니다. 다소 피로도가 높은 상황이나, 금일 중으로 예정된 주요 보고서 작성을 조속히 완료할 예정입니다."
  ➡️ 클립보드 자동 복사 및 공유 기능 연동
```

---

## 📅 로드맵 및 도전 과제
1. **커스텀 스타일 학습(Personalized Style)**: 사용자가 평소에 즐겨 쓰는 글 패턴을 분석하여, LLM이 사용자의 글쓰기 방식을 모방하는 나만의 스타일 복제 톤 연동.
2. **콘텐츠 길이 조절 유연성**: 프롬프트 지시어만으로 결과 텍스트의 문장 길이를 세밀하게 제어할 수 있는 가이드라인 프롬프트 보강.
