# ✈️ ¡Vamos a Jeju! AI Planner (시나리오 06) (로컬)

사용자의 다양한 여행 스타일(일정 기간, 동행자 유형, 선호 테마 카테고리, 선호 지역, 단답형 특별 요청 사항 등)을 반영하여, 제주도 100대 대표 관광지 데이터베이스에서 적합한 장소를 추천하고 지리적 동선 꼬임 없이 하루 일정을 설계해 주는 AI 기반 제주도 여행 플래너입니다.

---

## 📂 주요 기능 (Key Features)

1. **지리적 동선 최적화 (Geographical TSP Heuristic)**
   - 제주도를 동, 서, 남, 북 4대 권역으로 구분하여 하루 일정이 지리적으로 한쪽 권역에 묶이도록 기획합니다.
   - 제주 공항을 시발점으로 하여 다음 목적지까지의 거리를 반하버사인(Haversine) 공식으로 계산, 최단 거리 순으로 방문 순서를 자동 정렬(TSP 탐욕 알고리즘)하여 여행 시간을 아껴줍니다.

2. **하이브리드 AI 모드 토글 (Hybrid AI Execution)**
   - **Offline Heuristic Mode (기본)**: API 연동 없이 즉각 실행되며, 100% 로컬 계산을 통해 고품질의 여행 코스 및 예상 이동 시간을 생성합니다.
   - **Google Gemini 1.5 Flash**: API 키 등록 시 사용자의 자유로운 텍스트 요청(예: "동백꽃 위주로 돌고 흑돼지 맛집 가고 싶어요")을 상세하게 이해하여 명소 선정 및 묘사를 더 유연하고 감성적으로 처리합니다.
   - **Ollama Local LLM**: 로컬 PC에 구동 중인 Ollama API 환경(`gemma4:e2b` 등)을 토글하여 프라이버시가 완벽히 보호되는 플랜을 구상할 수 있습니다.

3. **100대 제주 관광지 기본 데이터베이스 탑재**
   - 자연(Nature), 힐링(Healing), 맛집/카페(CafeFood), 레저/체험(Activity), 역사/문화(Culture) 카테고리별로 정확히 100개의 명소를 내장하고 있습니다.
   - 각 장소의 위경도 정보와 카카오맵 고유 검색 태그가 연동되어 있습니다.

4. **카카오맵(KakaoMap) 길찾기 및 네비게이션 연동**
   - 생성된 일정표에서 '카카오맵 길찾기' 버튼을 클릭하면, 모바일 디바이스에 설치된 카카오맵 앱(kakaomap://) 또는 모바일 웹 브라우저가 즉각 실행되어 렌터카/대중교통 길찾기로 곧바로 연결됩니다.

## 🎨 디자인 시스템 & UI/UX (Aesthetics)

- **Obsidian Space Theme**: 심우주를 형상화한 어두운 네온 톤 배경위에 제주 감성 오렌지 오라(Orange Glow)와 청정 바다 터쿼이즈 오라(Turquoise Glow)가 유기적으로 흔들리도록 애니메이션 효과를 주었습니다.
- **Glassmorphism Design**: 모든 폼 컨트롤, 칩, 입력창, 일정 표시 카드가 반투명하고 얇은 백색 보더와 가벼운 블러를 가지는 프로스트 글라스 스타일로 구현되어 있어 매우 프리미엄하고 모던한 느낌을 줍니다.
- **Multi-Step Form Wizard**: 한 번에 너무 많은 정보를 입력받아 사용자 피로도를 유발하지 않도록 단계별 슬라이딩 애니메이션이 들어간 4단계 스타일 퀴즈를 제공합니다.

## 🛠️ 시작하기 (Quick Start)

1. `06_jeju_planner` 디렉토리로 이동합니다.
   ```bash
   cd 06_jeju_planner
   ```
2. 의존성 패키지를 내려받습니다.
   ```bash
   flutter pub get
   ```
3. 정적 분석을 통해 이상이 없는지 검토합니다.
   ```bash
   flutter analyze
   ```
4. 앱을 구동합니다.
   ```bash
   flutter run
   ```
